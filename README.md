# 🔢 Sudoku Solver (Python GUI)

A modern Sudoku Solver built using **Python + Tkinter**, featuring a **dark mode UI**, **file input**, and **real-time solving animation**.

---

## 🚀 Features

* 🎨 Dark Mode UI
* ⚡ Backtracking Algorithm
* 🎥 Visual Solving Animation
* 📂 Load Sudoku from File
* ✍️ Manual Input Support

---

## 🛠️ Tech Stack

* Python
* Tkinter (GUI)
* Backtracking Algorithm
